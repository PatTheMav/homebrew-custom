# Patthemav Custom

## How do I install these formulae?
`brew install patthemav/custom/<formula>`

Or `brew tap patthemav/custom`, followed by `brew trust patthemav/custom`. Contained formulas can then be installed via `brew install <formula>`.

## Documentation
`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
